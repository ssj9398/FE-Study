- 이도저도 아니다? 그럼 div를 써주자

- chrome스토어에 아웃라인 잡아주는 도구가 있다. 우선 연습 해보고 쓰도록 하자.   html5 outliner

2. section태그
- 넘버링(h1,h2~~) 새출발 하고 싶다 그럼 section을 쓰자 h1부터 다시 시작
- section은 nav, aside, article이 있다.
- nav는 section이긴한데 의미를 부여한 section
- 링크들을 전부 nav로 바꿔준다.
